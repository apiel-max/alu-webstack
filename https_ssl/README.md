# ALU System Engineering & DevOps

This repository contains projects completed as part of the System Engineering
& DevOps specialization at African Leadership University (ALU). Each project
focuses on a different layer of web infrastructure, building progressively
on the ones before it.

## Projects

### web_server
Installing and configuring Nginx on a single Ubuntu server: serving static
pages, setting up redirects, and building custom error pages — all via
automated Bash scripts.

### load_balancer
Scaling to two web servers behind an HAProxy load balancer, using
round-robin distribution and a custom HTTP response header to trace which
server handled each request.

### https_ssl
Securing the web stack with HTTPS: configuring DNS subdomains, terminating
SSL at the load balancer using HAProxy and Let's Encrypt/Certbot, and
enforcing HTTPS by redirecting all HTTP traffic.
